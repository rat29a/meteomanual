## 1. SYNOPTICKÁ SITUACE

**Synoptická situace** = rozložení tlakových útvarů, front a proudění nad Evropou.

### 🔍 Co hledám?
- Fronty
- Tlakové níže
- Tlakové výše
- Tlakové gradienty
- Vpád studeného / teplého vzduchu
- Stabilitu předpovědi
- Shoda modelů

### 📍 Kde to hledám?
- Mapy Evropy
- Tlakové mapy
- Modely

### 🔗 Zdroje
- [CHMI synoptická situace](https://intranet.chmi.cz/aktualni-situace/aktualni-stav-pocasi/evropa/synopticka-situace)
- [Wetter3 animace](https://www.wetter3.de/animation_en.html)
- Windy (tlak, izobary, vítr > zapnutým zobrazením izobar a pohybu částic)

### 💡 Co to znamená?

| Situace | Význam | Akce pro event |
|---------|--------|----------------|
| Fronta | Vyšší šance změny počasí | Připrav varianty, hlídej čas přechodu |
| Hluboká níže | Déšť, vítr, nestabilita | Kontrola kotev, drenáží, krytých zón |
| Výše | Klidnější počasí | Hlídej ranní mlhy, inverze |
| Těsný gradient | Silný vítr | Posil kotvení, omez lehké konstrukce |
| Shoda modelů | Vyšší jistota | Můžeš plánovat konkrétněji |
| Rozcházející se modely | Vyšší nejistota | Připrav více scénářů |

## 3. BOUŘKY & KONVEKCE

### 🔍 Co hledám?
- CAPE
- CIN
- Lifted Index
- K-index
- Střih větru (shear)
- Čas spuštění konvekce

### 📖 Vysvětlení

**CAPE** – Energie pro vznik bouřek
```
0–200 J/kg      = minimální
200–800 J/kg    = slabé přeháňky
800–1500 J/kg   = bouřky možné
1500–2500 J/kg  = silné bouřky
2500+ J/kg      = velmi nebezpečné prostředí
```

**CIN** – Blokace výstupu vzduchu
```
0 až -25 J/kg   = malá blokace
-25 až -75 J/kg = slabá
-75 až -150 J/kg = střední
-150+ J/kg      = silná (čeká na spouštěč)
```

**Lifted Index**
```
0 až -2   = slabá nestabilita
-3 až -5  = bouřky možné
-6 a méně = silné bouřky
```

**K-index**  
Rychlý ukazatel vhodnosti prostředí pro bouřky.

**Shear** – Změna větru s výškou. Důležité pro organizované bouřky.
- Slabý = běžné buňky < 15 m/s
- Střední = organizované bouřky 15-20m/s
- Silný = supercely / squall line možné > 20m/s

### 🔗 Zdroje
- Windy
- Meteologix
- ICON / GFS / Aladin
- Estofex
- Modelové soundingy

### 💡 Co to znamená?

| Situace | Význam | Akce pro event |
|---------|--------|----------------|
| Vysoké CAPE + nízké CIN | Rychlý vznik bouřek | Zkrácený monitoring, připravené kryty |
| Vysoké CAPE + shear | Silné, dlouho živé bouřky | Aktivuj plný bouřkový protokol |
| Vysoké CIN | Potenciál „čeká" na spouštěč | Hlídej frontu, orografii, denní ohřev |
| K-index >35 + vlhká spodní vrstva | Vysoká šance odpoledních přeháněk | Připrav stínění, vodu, drenáže |

## 4. SOUNDING / EMAGRAM – PRAKTICKÝ VÝKLAD

### 🔍 Co hledám?
- T a Td blízko sebe
- LCL
- Inverzi
- Suché vrstvy
- Strmý profil
- Nulovou hladinu
- PWAT
- DCAPE

### 📖 Vysvětlení pojmů

| Zkratka | Význam | Proč to řeším |
|---------|--------|---------------|
| **T** | Teplota vzduchu | Určuje stabilitu, typ srážek |
| **Td** | Rosný bod = teplota, kdy začne kondenzace | Kondenzace, vlhkost, mlha |
| **LCL** | Výška základny oblaků | Nízké = nízké základny, vysoké = sušší vzduch |
| **PWAT** | Množství vody v atmosféře | Vysoké = přívalový déšť |
| **DCAPE** | Riziko silných sestupných proudů | Vysoké = nárazy větru |
| **Nulová hladina** | Výška 0 °C | Rozhoduje o typu srážek |

### 🌤️ INTERPRETAČNÍ PŘÍRUČKA: „KDYŽ VIDÍM… TAK…"

#### ☁️ Mraky / oblačnost
**Hledej:** Místo, kde se teplota a rosný bod k sobě přiblíží nebo dotknou.

| Situace | Význam | Akce pro event |
|---------|--------|----------------|
| Malý rozdíl (0–2 °C) | Vzduch je téměř nasycený | Očekávej oblačnost, možná mlha |
| Dotyk ve výšce | Vrstva oblačnosti velmi pravděpodobná | Stínění bude proměnlivé |
| Dotyk ve výšce 2–6 km | Střední oblačnost | Minimální vliv na akci |
| Dotyk nad 6 km | Vysoká oblačnost | Šedý strop, možné slabé srážky |
| Dotyk při zemi | Mlha nebo nízká oblačnost | Riziko viditelnosti, vlhkost techniky |

#### 🌧️ Déšť / srážky
**Hledej:** Hloubku a sílu nasycené vrstvy. Nestačí jen vlhkost, důležitá je hloubka nasycené vrstvy.

| Situace | Význam | Akce pro event |
|---------|--------|----------------|
| Silná a hluboká vrstva, kde jsou křivky blízko sebe | Pravděpodobné srážky | Kontrola drenáží, pláštěny, kryté zóny |
| Vlhko jen v tenké vrstvě | Spíš bez srážek nebo slabé mrholení | Minimální opatření |
| Nad suchá vrstva a pod ní vlhko | Srážky se mohou vypařovat (virga) | Možný nárazový vítr při dopadu vzduchu |
| Hluboká vlhká vrstva (od země do 4+ km) | Trvalé/střední srážky | Připrav čerpadla, alternativní trasy |

#### ⚡ Bouřky / přeháňky
**Hledej:** Nestabilitu atmosféry.

| Situace | Význam | Akce pro event |
|---------|--------|----------------|
| Teplota rychle klesá s výškou | Nestabilita = potenciál konvekce | Sleduj CAPE/CIN, připrav stop-protokol |
| Spodní vrstvy teplé a vlhké | Palivo pro bouřky | Zkrácený monitoring |
| Velký CAPE (energie pro konvekci) | Silné bouřky možné | Aktivuj bouřkový protokol |
| Malá CIN (malá zábrana výstupu) | Rychlý vznik | Připravené kryty, evakuační trasy |
| Suchá vrstva nad vlhkou + vysoké DCAPE | Riziko downburstu | Kotvení konstrukcí, varování před nárazy |

#### ❄️ Sníh
**Hledej:** Kde je 0 °C hladina.

| Profil teplot | Typ srážek | Akce pro event |
|--------------|------------|----------------|
| Celý profil od mraku k zemi pod 0 °C | Sníh | Klouzavé povrchy, zátěž na střechy |
| Ve výšce nad nulou, u země pod nulou | Déšť se sněhem / mrznoucí déšť / ledovka | **Kritické** – led na kabelech, plochách |
| Jen horní část pod nulou | Déšť | Standardní dešťová opatření |

#### 🌫️ Mlha
**Hledej:** Když jsou teplota a rosný bod u země téměř stejné.

| Podmínky | Typ mlhy | Akce pro event |
|----------|----------|----------------|
| Jasná noc + slabý vítr + nasycení | Radiační mlha | Ranní zpoždění, osvětlení, informační tabule |
| Při advekci vlhkého vzduchu | Advekční mlha | Trvalejší omezení viditelnosti, bezpečnostní zóny |

#### 🔄 Inverze
**Hledej:** Vrstva, kde teplota s výškou roste místo klesání.

| Dopad | Význam | Akce pro event |
|-------|--------|----------------|
| Drží nízkou oblačnost a smog | Šedý strop, vlhko | Očekávej pomalé dissipace, připrav osvětlení |
| Brání konvekci | Snížené riziko bouřek | Může být „past": energie se uvolní najednou při průrazu |
| Omezuje vznik bouřek | Stabilní vrstva | Ventilace stanů, omezení generátorů |

### 🔗 Zdroje
- [CHMI aerologie](https://www.chmi.cz/letectvi/aerologicka-mereni?zalozka=radiosondazni)
- Windy sounding (Aladin)
- Meteologix
- Modelové soundingy

### ⚡ RYCHLÝ CHECK SOUNDINGU (30 vteřin)
1. **T a Td u sebe při zemi?** → Mlha / nízká oblačnost  
2. **Hluboká vlhká vrstva?** → Trvalejší srážky  
3. **Strmý profil + vlhké spodní vrstvy?** → Nestabilita, bouřkový potenciál  
4. **Suchá vrstva nad vlhkou + vysoké DCAPE?** → Riziko nárazového větru  
5. **Kde je 0 °C hladina?** → Typ srážek (déšť / sníh / ledovka)  
6. **Je tam inverze?** → Stabilní vrstva, omezená konvekce, riziko akumulace znečištění

## 5. SATELITNÍ DATA 🛰️

### 🔍 Co hledám?
- Oblačnost
- Vývoj systémů
- Vlhkostní pásy
- Proudění v horní troposféře
- Noční ochlazování
- Mlhy

### 📖 Typy snímků

**Visible (VIS)**  
Denní snímky, rozlišení struktury oblaků, mlha, sníh, hranice front.

**Infrared (IR)**  
Teplota vrcholů oblaků, intenzita konvekce (čím chladnější vrchol, tím vyšší a silnější bouřka).

**Water Vapor (WV)**  
Vlhkostní proudy, jet stream, suché vs. vlhké vzduchové hmoty, pohyb front.

### 🔗 Zdroje
- EUMETSAT (Meteosat)
- Windy satelitní vrstvy
- CHMI satelitní archivy
- RAMMB / Geo2K (globální přehled)

### 💡 Co to znamená?

| Situace | Význam | Akce pro event |
|---------|--------|----------------|
| Chladné IR vrcholy (< -50 °C) | Silná konvekce, možná krupobití/vichřice | Aktivuj bouřkový protokol, připrav kryty |
| WV tmavé pruhy | Suchý vzduch, může potlačit bouřky nebo zesílit downburst | Sleduj nárazový vítr |
| VIS mlha/stratus ráno | Advekční nebo radiační, očekávej pomalé dissipace při slabém větru | Informační tabule, osvětlení |
| Organizovaná linie v IR/WV | Squall line, rychlý postup 40–80 km/h | Evakuační protokol dle rychlosti pohybu |

### ⚡ Akce pro event
- Při přechodu chladného WV pásu + IR vrcholů → zkrácený nowcasting, připrav stop do 20–30 min
- Při satelitní lince směřující k areálu → aktivuj evakuační protokol dle rychlosti pohybu

## 7. LIMITNÍ HODNOTY & ROZHODOVACÍ MATICE

### 🌬️ Vítr
```
🟢 30 km/h      = běžný provoz
🟡 40 km/h      = lehké komplikace, zkontroluj plachty
🟠 60 km/h      = riziko stanů, omez činnost
🔴 80 km/h      = vysoké riziko techniky, stáhni konstrukce
⚫ 100+ km/h    = krizový stav, evakuace otevřených ploch
```

### 🌧️ Déšť
```
🟢 1 mm/h       = slabý
🟡 5 mm/h       = nepříjemný, připrav krytí
🟠 10 mm/h      = problém, kontrola drenáží/podlah
🔴 20 mm/h      = silný, omez pohyb, připrav čerpadla
⚫ 30+ mm/h     = přívalový, evakuace nížin/podjezdů
```

### 🌡️ Horko
```
🟢 28 °C        = zvýšená zátěž
🟡 30 °C        = nutná voda/stínění
🟠 32 °C        = vysoká zátěž, zkracuj směny
🔴 35+ °C       = kritické, aktivuj lékařský dohled
```

### ⚡ Bouřky & Blesky
```
🟢 >20 km       = sleduj
🟡 10–20 km     = příprava, informuj tým
🔴 <10 km       = STOP otevřených ploch, vést do krytů
🟢 30 min od posledního = možný návrat, kontrola stavu
```

**Pravidlo 30/30 pro blesky:**
- Pokud vidíš blesk a do 30 s uslyšíš hřmění → blesk je <10 km → okamžitá ochrana.
- Po posledním hřmění čekej 30 min před návratem.

## 8. MIKROKLMA, TERÉN & LOKÁLNÍ RIZIKA

### 🏔️ Topografie
- **Údolí** → drží chlad/mlhu, pomalejší dissipace
- **Hřebeny** → zesilují vítr, exponovaná místa
- **Svahy** → urychlují odtok vody, riziko eroze

### 💧 Vodní plochy
- Zvyšují vlhkost
- Podporují konvekci odpoledne
- Mění směr větru (břízový efekt)

### 🏙️ Urban / heat island
- Vyšší noční teploty
- Zesílená konvekce nad městy
- Turbulentní proudění mezi budovami

### 🌱 Půda / nasycení
- **Suchá půda** → rychlý odtok, eroze
- **Nasycená půda** → přetékání, kluzké povrchy, riziko pádů

### 🌳 Vegetace
- **Stromy** = bleskojisté body + riziko pádu větví při >50 km/h
- **Keře** = bariéra pro vítr, ale i past pro odtok

### ⚡ Akce pro event
- Mapuj únikové cesty, vyvýšená místa, kryté zóny, místa s výskytem vody
- Při nasycené půdě + předpovědi >10 mm/h → připrav čerpadla, sandbags, alternativní trasy
- Při silném větru + stromy >15 m → zkontroluj statiku, připrav záložní ukotvení

## 9. 30 SEKUNDOVÝ CHECK & OPERAČNÍ PROTOKOL

### 🔍 Co hledám?
- Radar?
- Blesky?
- Vítr?
- Trend?
- Nová buňka?
- Satelitní vrcholky?
- Tlak?

### 📍 Kde?
- Radar
- Lightning map
- Stanice
- Satelit
- Obloha
- METAR

### ⏱️ Postup (každých 15–30 min při hrozbě)
1. Podívej se na radar → směr, intenzita, rychlost
2. Zkontroluj blesky → vzdálenost, frekvence
3. Ověř vítr & tlak na nejbližší stanici → trend
4. Porovnej se satelitem → vývoj systémů
5. Rozhodni dle matice → 🟢 / 🟡 / 🟠 / 🔴 / ⚫
6. Komunikuj: čas, stav, akce, odpovědná osoba

### 📡 Komunikační řetězec
```
Meteorolog → Event Manager → Bezpečnostní tým → Technika → Zákazníci/Veřejnost
```

**Vždy uveď:**  
`Čas | Stav | Důvod | Akce | Další kontrola za X min`

### 📝 Příklad komunikace
```
14:30 | 🟠 ORANŽOVÁ | Bouřka 15 km severozápad, postup 40 km/h | 
Příprava krytů, informovat tým | Kontrola za 15 min
```
